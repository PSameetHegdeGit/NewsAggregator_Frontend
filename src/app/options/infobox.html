Information on the Different Parameters

Search Parameters:

Put a list of keywords. Note: separate each keyword with comma+space. If you put multiple keywords, will display with articles that have all keywords


Sources:

Select sources that you want to choose from. Currently support only those in dropdown


Language:

Only selects news sources of that language


What to Get:

Get Top Headlines

Gets headlining news articles with entered keywords from sources

Get Everything

Gets all news articles with those key words from the selected news sources
